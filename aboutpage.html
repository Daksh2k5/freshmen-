About

Welcome to Freshmen++!

This website has been developed using Freshmen23 as a reference for improvement and inspiration. This is just an effort to enhance and refine the original concept.

In the source code I noticed that several professors had reviews that could not be searched due to errors in the index. Additionally, I observed that new reviews were not being added to the database. Consequently, I decided to create this website.

I made a new website instead of fixing the original open source project so that I could add my own forms for submitting responses and to avoid interfering with the original project.

This website fixes the search system and the index with reviews, features a dark theme and aims to update the reviews more frequently. You are encouraged to visit freshmen23 for the authentic experience and to support the original creators.

If you encounter any problems regarding this website, or if the original creators want me to take this down, feel free to contact me.
